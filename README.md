# Machine Learning Classification Projects

This repository contains two supervised machine learning notebooks focused on distance-based classification methods.

## Projects

### K-Nearest Neighbor Classification

This notebook implements a K-Nearest Neighbor classifier from scratch and applies it to linear and nonlinear classification problems. It includes train-validation splitting, accuracy evaluation, hyperparameter tuning for $k$, and visualization of decision boundaries.

### Nearest Centroid Classification

This notebook implements a Nearest Centroid Classifier from scratch and applies it to synthetic datasets with different geometric structures. It demonstrates how class centroids are computed and how samples are assigned to the nearest centroid.

## Methods and Concepts

- Supervised classification
- K-Nearest Neighbor
- Nearest Centroid Classification
- Train-validation split
- Accuracy evaluation
- Hyperparameter tuning
- Decision boundary visualization
- Python, NumPy, scikit-learn, matplotlib
